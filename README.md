# tbFTP

# Secure File Transfer Protocol (sFTP)

A lightweight, secure file transfer tool over HTTP using hybrid encryption, token-based authentication, and digital signatures. Designed for safe communication in low-tech environments, with simple CLI and server interface.

## Features

- Hybrid encryption (AES + RSA)
- Digital signature with sender verification
- User registration and token-based authentication
- CLI for registration and file sending
- Flask-based HTTP server to receive and decrypt files
- Built-in protection against tampering and replay attacks

- Python 3.8+
- pip (Python package manager)
